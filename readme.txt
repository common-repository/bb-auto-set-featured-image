This plugin automatically sets featured image for your blog posts if post thumbnail is not set manually. This extracts keywords from your article title and content and searches for image in pixabay.com. 

== Description ==

This plugin automatically sets featured image for your blog posts if post thumbnail is not set manually. This extracts keywords from your article title and content and searches for image in pixabay.com. 


== Installation ==

Upload the BB Auto Set Featured Image plugin to your blog, Activate it

You're done!
